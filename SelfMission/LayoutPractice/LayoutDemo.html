实现一个两栏布局，左侧固定宽度，右侧根据浏览器宽度进行自适应变化

使用float

1- 使用单左边float + 右边margin-left空出左边的Width
2- 使用双float
定宽200px; float: left;
计算剩余宽度calc(100% - 200px); float: left;

使用position

父容器relative
1- 子元素左侧定宽200px, absolute, top/left: 0; 右侧margin-left留出200px;
2- 或者反过来左侧定宽，右侧absolute, top: 0; left: 200px; 但是这种方案右侧元素不能占满剩下的空间。需再加上right: 0px;

使用flex

父容器display: flex
左边元素定宽200px; 右边元素flex: 1;表示占满剩下的全部空间

实现一个两栏布局，右侧固定宽度，左侧根据浏览器宽度进行自适应变化

使用float

左侧calc(100% - 200px)动态计算, float: left;
右侧定宽200px, float: right

使用position

父容器relative 左侧子元素 absolute, left/top: 0; right: 右侧元素的宽;
右侧元素 absolute,
top/right: 0;

使用flex

父容器flex 左侧子元素 flex: 1; 占满剩下的空间
右侧元素 width: 200px

实现一个三栏布局，左侧固定宽度，右侧固定宽度，中间部分宽度随浏览器宽度变化而自适应变化

使用float

float left; width: 200px;
float left; width: calc(100% - 400px);
float right; width: 200px;

使用position

父容器relative
left: absolute, width: 200px; top/left: 0;
(1) - margin-left, margin-right: 200px; middle不使用position
(2) - 使用position, absolute; left, right: 200px; top: 0;
right: absolute, width: 200px; left: 200px; top, right: 0;

使用flex

left: width: 200px;
middle: flex: 1;
right: width: 200px;

实现一个三栏布局，左侧固定宽度，中间固定宽度，右侧根据浏览器宽度变化而自适应变化

使用float

left和middle定宽200px, right使用float: left/right都可以
middle
(1) - left和middle改成inline-block, 并且在left这个div上设置margin-right: -4px; 因为默认inline-block之间会有4px的间距 最后再在inline-block的元素上添加: vertical-align: top;让他们顶头对齐
(2) - left和middle也使用float: left;

使用position

left和middle定宽200px
display: inline-block;
vertical-align: top;
middle
right, position: absolute, top, right: 0; left: 400px;

使用flex

middle
